To run either app make sure you first start the provided server.

To start the server, open a terminal window within the provided
PlaceJsonRPCServer folder and use the command:
java -jar lib/placeserver.jar 8080

Once the server is running you can unpack the apps and build/run 
them accordingly through xcode or android studio.

The contents of the provided places.json file should 
then appear and you can add/delete/edit and make calculations
accordingly.
